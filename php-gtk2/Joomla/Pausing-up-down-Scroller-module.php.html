Pausing up-down Scroller module

  • Last Updated: 22 June 2015
  • Hits: 173336

This is the module version of the Pausing up-down Scroller plugin.

This module allows you to display an up-down scroller that pauses between each message! The scroller is displayed in a module position. The look of the scroller is completely styled using external CSS, including the dimensions.

How to use the "Pausing up-down Scroller" module

The syntax of the scrolling message definition is exactly the same as that of the Pausing up-down Scroller plugin plugin.

Download

Very Important: You must install the latest version of the Pausing up-down Scroller plugin for this module to work.

Joomla 1.0.x: mod_pausing_up_down_scroller_v1.03.zip
Joomla 1.5.x: mod_pausing_up_down_scroller_v1.53.zip
Joomla 1.6.x: mod_pausing_up_down_scroller_v1.63.zip
Joomla 1.7.x: mod_pausing_up_down_scroller_v1.73.zip
Joomla 2.5.x: mod_pausing_up_down_scroller_v2.53.zip

Very Important: You must install the latest version of the Pausing up-down Scroller plugin for this module to work.

How to install
  1. First, make sure you have installed the Pausing up-down Scroller plugin. This module DOES NOT work without the Pausing up-down Scroller plugin.

  2. Download and install the Pausing up-down Scroller module.

  3. Make sure both the plugin and the module are published / enabled.

  4. Select a module position where you want to place the module.

  5. Go to Module Manager and click on "Pausing up-down Scroller module".

  6. In the parameter "Content", enter the scroller messages following the syntax as outlined here. For example,

    7. Important: If you're using Joomla 1.5.8, please refer to the article:
    How to enter HTML tags, javascript and PHP codes in Joomla 1.5.8?
    Otherwise you might find that all your HTML tags, Javascript and PHP scripts will be removed the moment you click the Save or Apply button.
